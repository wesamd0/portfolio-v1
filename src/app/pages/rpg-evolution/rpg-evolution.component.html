Écosystème Cross-Play Distribué & Mobile

Dart Flutter Socket.io Node.js Architecture Distribuée AWS (EC2) Ingénierie des exigences (SRS) CI/CD

Transformation d'une architecture monolithique Web en un système distribué multi-clients. Orchestration de la synchronisation temps réel entre une app Mobile (client léger) et un client Desktop (client lourd) existant, avec gestion de la dette technique.

Le Challenge Technique

Transformer la plateforme Web en un produit commercialisable multiplateforme. Au-delà de l'architecture distribuée, le défi majeur réside dans la refonte complète de l'interface (UI/UX) pour adapter les contrôles Desktop au format Mobile. Cela implique l'intégration de nouvelles fonctionnalités dans le respect rigoureux du cahier des charges (SRS) et des contraintes strictes de maintenance logicielle.

Responsabilité Système & Intégration

  • Architecture & Cloud : Solution distribuée sur AWS synchronisant les clients hétérogènes via Socket.io.
  • Développement Mobile : Transposition de l'UX Desktop vers une interface tactile intuitive (Dart) intégrant les capteurs natifs.
  • Ingénierie : Pilotage des exigences (SRS) et gestion budgétaire simulée (~1000h).
  • Intégration : Implémentation de nouvelles fonctionnalités dans le code existant (Legacy) en assurant la non-régression.
