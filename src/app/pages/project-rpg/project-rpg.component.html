RPG Tactique Multijoueur

Angular 17 TypeScript Node.js / Express MongoDB WebSockets AWS (EC2) CI/CD (GitLab) Jasmine / Jest / Mocha / Chai

Conception d'une application Full Stack (Angular/Node.js) de jeu multijoueur, intégrant une synchronisation temps réel (WebSockets) et un pipeline de déploiement continu (CI/CD).

Le Challenge Technique

Développer une application Full Stack robuste capable de gérer des parties multijoueurs. Le défi était d'orchestrer une architecture distribuée complète. Il fallait non seulement gérer la synchronisation multijoueur temps réel (WebSockets), mais aussi automatiser le déploiement (CI/CD) du serveur sur AWS et du client sur GitLab Pages sans briser la compatibilité. Il fallait également tester et valider rigoureusement l'ensemble du système à travers des tests unitaires et valider la qualité du code via des revues de code.

Contribution Full Stack (Projet Agile)

Au sein d'une équipe Scrum, j'ai participé activement à l'ensemble du cycle de développement, sans me limiter à une seule technologie :

  • Développement Transversal : Implémentation de fonctionnalités complètes, de l'interface Angular jusqu'à la logique serveur Node.js et la base de données.
  • Assurance Qualité : Rédaction de tests unitaires (Jest/Mocha/Chai) et validation croisée du code via des revues de code (Merge Requests) rigoureuses.
  • DevOps & Architecture : Configuration du pipeline CI/CD et participation aux décisions d'architecture (modélisation UML, API REST).
